#room with no view

##game by andrew ksendzoff

###about:

room with no view is a text based adventure game, like zork. as a player, you can
interact with the environment in different ways, such as moving from place to place,
picking up items, and using them on various parts of the environment.

###commands:

goto [place]: goes to place as long as you can currently see it. can also be used
to go to cardinal directions (north, south, east, west).

look [object]: gives player info about the object.

pickup [object]: lets player pick up object.
